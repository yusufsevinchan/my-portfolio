2048 Oyunu (WinForms Uygulaması)

Bu proje, klasik 2048 oyununu WinForms kullanarak geliştirdiğim bir masaüstü oyunudur. Oyun, kullanıcının sayı bloklarını birleştirerek 2048'e ulaşmaya çalıştığı bir puzzle oyunudur.

Proje, C# programlama dili kullanılarak geliştirilmiştir. Oyunun arayüzü, Windows Forms uygulamaları için tasarlanmış olan Visual Studio arayüzü kullanılarak tasarlanmıştır.

Kullanılan teknolojiler:

  • C#
  • WinForms
  • Visual Studio

Hava Durumu Chrome Eklentisi

Bu proje, Chrome tarayıcısı için geliştirdiğim bir hava durumu eklentisidir. Eklenti, kullanıcının bulunduğu konumu otomatik olarak alır ve anlık hava durumu bilgilerini gösterir.

Proje, JavaScript ve HTML/CSS kullanılarak geliştirilmiştir. Hava durumu verileri, bir hava durumu API'si kullanılarak alınmaktadır.

Kullanılan teknolojiler:

  • JavaScript
  • HTML/CSS
  • Open Weather Map API
  • Chrome Eklentisi

Blazor Admin panel

Bu proje, Blazor Server ve Blazor WebAssembly kullanarak geliştirdiğim bir admin panelidir. Admin paneli, Kullanıcıların, Kullanıcı Rollerinin, Kursların, Kurs Kategorilerinin ve Eğitmenlerin yönetildği bir yönetim panelidir.

Proje, C# programlama dili kullanılarak .NET 8 platformunda geliştirilmiştir. Admin paneli, Blazor Server ve Blazor WebAssembly teknolojileri kullanılarak geliştirilmiştir. Bootstrap ve CSS kullanılarak tasarlanmıştır.

Kullanılan teknolojiler:

  • C#
  • Blazor Server
  • Blazor WebAssembly
  • Entity Framework
  • SQL
  • Visual Studio
  • Bootstrap
